# 木更津市立木更津第三中学校同窓会（21卒）特設サイト

令和8年（2026年）1月11日に開催される、木更津第三中学校（2021年3月卒業生）同窓会のための案内用Webサイトです。

## 公開URL
https://mogotyanasuka1212-maker.github.io/reunion-2025-hp/

## サイトの概要
* **目的**: 同窓会の開催日時、会場、会費、注意事項の周知
* **対象**: 2021年3月卒業生および教職員
* **管理者**: 同窓会実行委員会（委員長：鈴木明輝）

## ファイル構成
* `index.html`: トップページ（開催概要、注意事項など）
* `member.html`: 実行委員メンバー紹介
* `venue.html`: 会場（舵輪）の紹介・地図
* `contact.html`: お問い合わせ案内（メール）
* `style.css`: 全ページのデザイン設定

## 運用・更新方法（自分用メモ）

### 内容を修正したいとき
1.  VSCodeで該当するファイル（`index.html`など）を開いて文字を修正する。
2.  保存する（Ctrl + S）。
3.  左側の「ソース管理（枝分かれアイコン）」をクリックする。
4.  入力欄に「修正」や「文言変更」などメモを入力する。
5.  「コミット」ボタンを押す。
6.  「変更の同期（またはプッシュ）」ボタンを押す。
    * → 数分後にWebサイトに反映されます。

### 公開を終了するとき（同窓会終了後）
1.  GitHubのリポジトリページを開く。
2.  「Settings」タブをクリックする。
3.  ページ一番下の「Danger Zone」にある「Delete this repository」を選択する。
4.  画面の指示に従って削除を実行する。
    * → サイトもファイルもすべて削除されます。
